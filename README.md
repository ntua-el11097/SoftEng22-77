## Επισκόπηση

Το παρόν έργο αποτελεί ένα full-stack, fully-documented project που πραγματοποιήθηκε στα πλαίσια του μαθήματος "Τεχνολογία Λογισμικού" της Σχολής Ηλεκτρολόγων Μηχανικών και Μηχανικών Υπολογιστών του Εθνικού Μετσόβιου Πολυτεχνείου.

Το αντικείμενο της εργασίας είναι η ανάπτυξη μιας εφαρμογής λογισμικού για τη διαμόρφωση και εκτέλεση "έξυπνων ερωτηματολογίων". Ως "έξυπνο", χαρακτηρίζεται ένα ερωτηματολόγιο όταν η κάθε επόμενη ερώτηση και οι απαντήσεις της δύναται να καθορίζεται από την απάντηση της προηγούμενης.

## Directories

Το παρόν repository περιλαμβάνει τα εξής directories:
| Directory | Σύνοψη |
| --------- | ---------|
| api-backend | Το API και το Back-end του λογισμικού, υλοποιημένα με nodeJS και express |
| front-end | Το Front-end του λογισμικού, υλοποιημένο με ReactJS |
| test | Το testing του API, υλοποιημένο με το εργαλείο Postman |
| vpp | Το documentation του λογισμικού, υλοποιημένο με το εργαλείο Visual Paradigm. Επιπλέον, εδώ βρίσκεται και το documentation του API, υλοποιημένο με το εργαλείο Postman |
| data | Database dumps και κάποια sample questionnaires. Χρήσιμα για το testing |

## Προαπαιτούμενα

Για την εγκατάσταση και λειτουργία της εφαρμογής, απαιτούνται τα nodeJS και npm.
Ως σύστημα διαχείρισης βάσης δεδομένων χρησιμοποιείται το MongoDB v4.2.
Για το documentation, απαιτείται το Visual Paradigm.
Τέλος, για το testing απαιτείται επιπλέον το εργαλείο Postman.

Πληροφορίες για την εγκατάσταση και λειτουργία του κάθε μέρους, βρίσκονται στα README των αντίστοιχων directories.
