REST Service - Get questionnaire link

Properties
Name Value
Description

Επιστροφή object που περιέχει τα γενικά στοιχεία και τις ερωτήσεις του ερωτηματολογίου με αναγνωριστικό questionnaireID, ταξινομημένες ως προς το αναγνωριστικό της ερώτησης.

Εάν η προαιρετική query παράμετρος "format" λάβει την τιμή "csv", τότε η απάντηση έχει μορφή csv, και όχι JSON object.

Uri /questionnaire/:questionnaireID
Method GET
Resource Type Service Provider

questionnaireID path string true

format query string false
