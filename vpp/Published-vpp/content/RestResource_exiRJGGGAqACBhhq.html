Description

Καταχώρηση της απάντησης optionID η οποία δόθηκε στο γεγονός απάντησης session στην ερώτηση questionID του ερωτηματολογίου questionnaireID. Το αναγνωριστικό session είναι μια συμβολοσειρά με 4 τυχαίους χαρακτήρες που αντιστοιχούν στο γεγονός απάντησης του ερωτηματολογίου από κάποιον χρήστη.

Εάν η προαιρετική query παράμετρος "format" λάβει την τιμή "csv", τότε η απάντηση έχει μορφή csv, και όχι JSON object.

Uri /doanswer/:questionnaireID/:questionID/:session/:optionID
Method POST
